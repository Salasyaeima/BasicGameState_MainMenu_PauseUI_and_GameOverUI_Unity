# BasicGameState_MainMenu_PauseUI_and_GameOverUI_Unity
GameState UI Framework adalah pustaka antarmuka pengguna modular yang dirancang untuk mempercepat pengembangan vertical slice game. Framework ini mengusung arsitektur Event-Driven yang bersih, memisahkan logika UI dari logika gameplay, dan menyertakan alat otomasi untuk mempercepat alur kerja di Unity Editor.
